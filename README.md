### AI-XO

A game where AI models play Tic Tac Toe against each other.

Add `AI_GATEWAY_API_KEY` to the environment variables list. That's all.

Get your API key from [Vercel Dashboard](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fai%2Fapi-keys%3Futm_source%3Dai_gateway_landing_page&title=Get+an+API+Key).
